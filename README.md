# Statistic-exercise

## Week 9 : หาความน่าจะเป็นของเหตุการณ์

ตัวอย่าง : ความน่าจะเป็นที่นักกีฬาจะโยนลูกบาสลงห่วงมากกว่า 6 ครั้ง\

ในเรื่องนี้เราจะต้องรู้ให้ได้ว่า ตัวแปร X จะถูกเขียนยังไง อาจจะเป็น X > 1, X <= 5, 3 < X < 10 เพื่อไปหาพื้นที่ใต้กราฟต่อไป

## Week 10 : หาความน่าจะเป็นของผลต่าง ผลต่างสัดส่วน ความแปรปรวนที่กำหนด และความแปรปรวน 2 กลุ่ม

ตัวอย่าง 1: จงหาความน่าจะเป็นที่สวน A ให้ผลผลิตมากกว่าสวน B อยู่ 10 กิโลกรัม\
ตัวอย่าง 2: ความน่าจะเป็นที่จะมีคนเห็นด้วยในนโยบาย A น้อยกว่านโยบาย B อยู่ 50 คน\
ตัวอย่าง 3: ความน่าจะเป็นที่ความแปรปรวนในกระบวนการผลิตจะอยู่ระหว่าง 1.35 - 2.76 นาที\
ตัวอย่าง 4: ความแปรปรวนในเครื่องจักร A และ B มีโอกาสที่จะห่างกันเกิน 3 คือเท่าไร\

เป็นการดูว่าโอกาสที่เหตุการณ์จะเกิดมันเป็นไปได้เท่าไรเมื่อเทียบจากที่โจทย์กำหนดให้
