### แบบฝึกหัดบทที่ 9

#### [cite_start]**ข้อที่ 1: กรณีประชากรมีการแจกแจงปกติและทราบความแปรปรวน ($\sigma^2$)** [cite: 4]
**โจทย์:** จงหาความน่าจะเป็นที่ความสูงเฉลี่ยของนักเรียนชายที่สุ่มมา 49 คน จะมากกว่า 163 ซม. โดยกำหนดให้ความสูงของนักเรียนชายมีการแจกแจงปกติ มีค่าเฉลี่ย ($\mu$) = 160 ซม. [cite_start]และความแปรปรวน ($\sigma^2$) = 100 [cite: 4, 5]

**วิธีทำ:**
1.  [cite_start]**วิเคราะห์โจทย์:** ทราบ $\mu = 160, \sigma^2 = 100$ (ดังนั้น $\sigma = 10$), $n = 49$ [cite: 4]
2.  [cite_start]**เปลี่ยนตัวแปรสุ่ม $\bar{x}$ เป็น $Z$:** ใช้สูตร $Z = \frac{\bar{x} - \mu}{\sigma/\sqrt{n}}$ [cite: 4]
3.  [cite_start]**คำนวณค่า $Z$:** หา $P(\bar{x} > 163)$ จะได้ $P(Z > \frac{163 - 160}{10/\sqrt{49}}) = P(Z > \frac{3}{1.43}) \approx P(Z > 2.1)$ [cite: 5]
4.  [cite_start]**เปิดตาราง:** $P(Z > 2.1) = 1 - P(Z < 2.1) = 1 - 0.9821 = 0.0179$ [cite: 5]
[cite_start]**คำตอบ:** ความน่าจะเป็นคือ **0.0179** [cite: 5]

#### [cite_start]**ข้อที่ 2: กรณีประชากรแจกแจงปกติ ไม่ทราบ $\sigma^2$ แต่กลุ่มตัวอย่างขนาดใหญ่ ($n \ge 30$)** [cite: 5]
[cite_start]**โจทย์:** สุ่มตัวอย่างคนมา 50 คน พบว่ามีส่วนเบี่ยงเบนมาตรฐานของรายได้ ($s$) = 3,000 บาท หากประชากรมีรายได้เฉลี่ย ($\mu$) = 15,000 บาท จงหาความน่าจะเป็นที่ค่าเฉลี่ยของกลุ่มตัวอย่างจะมีรายได้เฉลี่ยอย่างน้อย 16,000 บาท [cite: 5, 6]

**วิธีทำ:**
1.  [cite_start]**วิเคราะห์โจทย์:** $\mu = 15000, s = 3000, n = 50$ [cite: 5]
2.  [cite_start]**เปลี่ยนตัวแปรสุ่ม $\bar{x}$ เป็น $Z$:** (เนื่องจาก $n$ ใหญ่ ใช้ $s$ แทน $\sigma$ ได้) $Z = \frac{\bar{x} - \mu}{s/\sqrt{n}}$ [cite: 5]
3.  [cite_start]**คำนวณค่า $Z$:** หา $P(\bar{x} \ge 16000)$ จะได้ $P(Z \ge \frac{16000 - 15000}{3000/\sqrt{50}}) = P(Z \ge 2.36)$ [cite: 6]
4.  [cite_start]**เปิดตาราง:** $1 - P(Z < 2.36) = 1 - 0.9909 = 0.0091$ [cite: 6]
[cite_start]**คำตอบ:** ความน่าจะเป็นคือ **0.0091** [cite: 6]

#### [cite_start]**ข้อที่ 3: กรณีประชากรแจกแจงปกติ ไม่ทราบ $\sigma^2$ และกลุ่มตัวอย่างขนาดเล็ก ($n < 30$)** [cite: 6]
**โจทย์:** สุ่มตัวอย่างน้ำยามา 9 ขวด มีค่าส่วนเบี่ยงเบนมาตรฐาน ($s$) = 6 มล. ถ้าปริมาตรสุทธิเฉลี่ยของประชากร ($\mu$) = 200 มล. [cite_start]จงหาความน่าจะเป็นที่ปริมาตรสุทธิเฉลี่ยของ 9 ขวดนี้จะไม่เกิน 195 มิลลิลิตร [cite: 6, 7]

**วิธีทำ:**
1.  [cite_start]**วิเคราะห์โจทย์:** $\mu = 200, n = 9, s = 6, df = n-1 = 8$ [cite: 6]
2.  [cite_start]**เปลี่ยนตัวแปรสุ่ม $\bar{x}$ เป็น $T$:** (เนื่องจาก $n$ เล็ก) $T = \frac{\bar{x} - \mu}{s/\sqrt{n}}$ [cite: 6]
3.  [cite_start]**คำนวณค่า $T$:** หา $P(\bar{x} \le 195)$ จะได้ $P(T \le \frac{195 - 200}{6/\sqrt{9}}) = P(T \le -2.5)$ [cite: 7]
4.  [cite_start]**เปิดตาราง $t$:** ที่ $df = 8$ จะได้ค่าความน่าจะเป็นเท่ากับ $0.0185$ [cite: 7]
[cite_start]**คำตอบ:** ความน่าจะเป็นคือ **0.0185** [cite: 7]

#### [cite_start]**ข้อที่ 4: กรณีประชากรมีการแจกแจงแบบอื่นๆ (ใช้ Central Limit Theorem)** [cite: 8]
[cite_start]**โจทย์:** สุ่มผู้ใช้โทรศัพท์ 35 ราย พบว่าเวลาเฉลี่ยที่ใช้คือ 120 วินาที มีส่วนเบี่ยงเบนมาตรฐาน ($s$) = 40 วินาที จงหาความน่าจะเป็นที่เวลาเฉลี่ยที่ใช้จะมากกว่า 140 วินาที [cite: 8]

**วิธีทำ:**
1.  [cite_start]**วิเคราะห์โจทย์:** $\mu = 120, s = 40, n = 35$ (โจทย์ไม่ระบุการแจกแจง แต่ $n \ge 30$ จึงใช้ $Z$ ได้ตามกฎ CLT) [cite: 8]
2.  [cite_start]**คำนวณค่า $Z$:** หา $P(\bar{x} > 140)$ จะได้ $P(Z > \frac{140 - 120}{40/\sqrt{35}}) = P(Z > 2.96)$ [cite: 8]
3.  [cite_start]**เปิดตาราง:** $1 - P(Z < 2.96) = 1 - 0.9985 = 0.0015$ [cite: 8]
[cite_start]**คำตอบ:** ความน่าจะเป็นคือ **0.0015** [cite: 8]

### **แบบฝึกหัดที่แต่งขึ้นใหม่ (ข้อที่ 5)**

**โจทย์:** โรงงานผลิตหลอดไฟแห่งหนึ่งระบุว่าหลอดไฟมีอายุการใช้งานเฉลี่ย ($\mu$) 800 ชั่วโมง และมีส่วนเบี่ยงเบนมาตรฐาน ($\sigma$) 40 ชั่วโมง หากสุ่มตัวอย่างหลอดไฟมา 64 หลอด จงหาความน่าจะเป็นที่อายุการใช้งานเฉลี่ยของหลอดไฟกลุ่มนี้จะอยู่ระหว่าง 790 ถึง 810 ชั่วโมง

**วิธีทำ:**
1.  **วิเคราะห์โจทย์:** $\mu = 800, \sigma = 40, n = 64$
2.  **หาความคลาดเคลื่อนมาตรฐาน ($SE$):** $\sigma_{\bar{x}} = \frac{\sigma}{\sqrt{n}} = \frac{40}{\sqrt{64}} = \frac{40}{8} = 5$
3.  **เปลี่ยนค่า $\bar{x}$ เป็น $Z$:**
    * สำหรับ $\bar{x} = 790$: $Z_1 = \frac{790 - 800}{5} = -2$
    * สำหรับ $\bar{x} = 810$: $Z_2 = \frac{810 - 800}{5} = 2$
4.  **หาความน่าจะเป็น $P(-2 < Z < 2)$:**
    * จากตาราง $Z$: $P(Z < 2) = 0.9772$ และ $P(Z < -2) = 0.0228$
    * ความน่าจะเป็นคือ $0.9772 - 0.0228 = 0.9544$

**คำตอบ:** ความน่าจะเป็นที่อายุการใช้งานเฉลี่ยจะอยู่ระหว่าง 790 ถึง 810 ชั่วโมง คือ **0.9544 (หรือ 95.44%)**
