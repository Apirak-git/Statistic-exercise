### ข้อที่ 1: กรณีประชากรปกติและทราบความแปรปรวน ($\sigma^2$)
**โจทย์:** จงหาความน่าจะเป็นที่ความสูงเฉลี่ยของนักเรียนชายที่สุ่มมา 49 คน จะมากกว่า 163 ซม. โดยกำหนดให้ความสูงของนักเรียนชายมีการแจกแจงปกติ มีค่าเฉลี่ย ($\mu$) = 160 ซม. และความแปรปรวน ($\sigma^2$) = 100

**วิธีทำ:**
1. **วิเคราะห์ข้อมูล:**
   - $\mu = 160$
   - $\sigma^2 = 100 \implies \sigma = 10$
   - $n = 49$
2. **คำนวณค่า $Z$:**
   $$Z = \frac{\bar{x} - \mu}{\sigma/\sqrt{n}} = \frac{163 - 160}{10/\sqrt{49}} = \frac{3}{1.428} \approx 2.1$$
3. **หาความน่าจะเป็น $P(\bar{x} > 163)$:**
   $$P(Z > 2.1) = 1 - P(Z < 2.1) = 1 - 0.9821 = 0.0179$$

**✅ คำตอบ:** ความน่าจะเป็นคือ **0.0179**

### ข้อที่ 2: กรณีกลุ่มตัวอย่างขนาดใหญ่ ($n \ge 30$) และไม่ทราบ $\sigma^2$
**โจทย์:** สุ่มตัวอย่างคนมา 50 คน พบว่ามีส่วนเบี่ยงเบนมาตรฐานของรายได้ ($s$) = 3,000 บาท หากประชากรมีรายได้เฉลี่ย ($\mu$) = 15,000 บาท จงหาความน่าจะเป็นที่ค่าเฉลี่ยของกลุ่มตัวอย่างจะมีรายได้เฉลี่ยอย่างน้อย 16,000 บาท

**วิธีทำ:**
1. **วิเคราะห์ข้อมูล:**
   - $\mu = 15,000$
   - $s = 3,000$
   - $n = 50$ (ใช้ $s$ แทน $\sigma$ ได้เพราะ $n$ ใหญ่)
2. **คำนวณค่า $Z$:**
   $$Z = \frac{16,000 - 15,000}{3,000/\sqrt{50}} = \frac{1,000}{424.26} \approx 2.36$$
3. **หาความน่าจะเป็น $P(Z \ge 2.36)$:**
   $$1 - P(Z < 2.36) = 1 - 0.9909 = 0.0091$$

**✅ คำตอบ:** ความน่าจะเป็นคือ **0.0091**

### ข้อที่ 3: กรณีกลุ่มตัวอย่างขนาดเล็ก ($n < 30$) และไม่ทราบ $\sigma^2$
**โจทย์:** สุ่มตัวอย่างน้ำยามา 9 ขวด มีค่าส่วนเบี่ยงเบนมาตรฐาน ($s$) = 6 มล. ถ้าปริมาตรสุทธิเฉลี่ยของประชากร ($\mu$) = 200 มล. จงหาความน่าจะเป็นที่ปริมาตรสุทธิเฉลี่ยของ 9 ขวดนี้จะไม่เกิน 195 มิลลิลิตร

**วิธีทำ:**
1. **วิเคราะห์ข้อมูล:**
   - $\mu = 200, n = 9, s = 6$
   - องศาอิสระ ($df$) = $9 - 1 = 8$
2. **คำนวณค่า $T$:**
   $$T = \frac{195 - 200}{6/\sqrt{9}} = \frac{-5}{2} = -2.5$$
3. **หาความน่าจะเป็น $P(T \le -2.5)$ จากตาราง t-distribution:**
   - เปิดตารางที่ $df = 8$ ได้ค่าความน่าจะเป็นพื้นที่หางเท่ากับ $0.0185$

**✅ คำตอบ:** ความน่าจะเป็นคือ **0.0185**

### ข้อที่ 4: การใช้ทฤษฎีบทเข้าสู่ส่วนกลาง (Central Limit Theorem)
**โจทย์:** สุ่มผู้ใช้โทรศัพท์ 35 ราย พบว่าเวลาเฉลี่ยที่ใช้คือ 120 วินาที มีส่วนเบี่ยงเบนมาตรฐาน ($s$) = 40 วินาที จงหาความน่าจะเป็นที่เวลาเฉลี่ยที่ใช้จะมากกว่า 140 วินาที

**วิธีทำ:**
1. **วิเคราะห์ข้อมูล:** $\mu = 120, s = 40, n = 35$
2. **คำนวณค่า $Z$:**
   $$Z = \frac{140 - 120}{40/\sqrt{35}} = \frac{20}{6.76} \approx 2.96$$
3. **หาความน่าจะเป็น $P(Z > 2.96)$:**
   $$1 - 0.9985 = 0.0015$$

**✅ คำตอบ:** ความน่าจะเป็นคือ **0.0015**

### ข้อที่ 5: การประยุกต์ใช้ในการควบคุมคุณภาพการผลิต
**โจทย์:** โรงงานผลิตหลอดไฟระบุว่าอายุการใช้งานเฉลี่ยของหลอดไฟ ($\mu$) คือ 800 ชั่วโมง และมีส่วนเบี่ยงเบนมาตรฐาน ($\sigma$) 40 ชั่วโมง หากฝ่ายประกันคุณภาพสุ่มหลอดไฟมาทดสอบจำนวน 64 หลอด จงหาความน่าจะเป็นที่ค่าเฉลี่ยอายุการใช้งานของกลุ่มตัวอย่างนี้จะ **อยู่ระหว่าง 790 ถึง 810 ชั่วโมง**

**วิธีทำ:**
1. **วิเคราะห์ข้อมูล:**
   - $\mu = 800$
   - $\sigma = 40$
   - $n = 64$
2. **คำนวณความคลาดเคลื่อนมาตรฐาน (Standard Error):**
   $$SE = \frac{\sigma}{\sqrt{n}} = \frac{40}{\sqrt{64}} = \frac{40}{8} = 5$$
3. **แปลงค่า $\bar{x}$ เป็นค่า $Z$ (2 ค่า):**
   - ที่ $\bar{x} = 790$: $Z_1 = \frac{790 - 800}{5} = -2$
   - ที่ $\bar{x} = 810$: $Z_2 = \frac{810 - 800}{5} = +2$
4. **หาพื้นที่ภายใต้เส้นโค้งปกติ $P(-2 < Z < 2)$:**
   - $P(Z < 2) = 0.9772$
   - $P(Z < -2) = 0.0228$
   - พื้นที่ระหว่างกลาง = $0.9772 - 0.0228 = 0.9544$

**✅ คำตอบ:** ความน่าจะเป็นที่อายุการใช้งานเฉลี่ยจะอยู่ระหว่าง 790 ถึง 810 ชั่วโมง คือ **0.9544** (หรือคิดเป็น **95.44%**)
